How to Use This Tool

Left Panel — Pipeline Detections

Shows bounding boxes from the ML pipeline over the Sanborn map. Color indicates confidence:
Green ≥ 0.80 Yellow 0.65–0.79 Red < 0.65 Orange = OCR noise Purple = Female Boarding (full phrase)
Each box shows the detected text. Both panels show the same clean map image.

Right Panel — Marking

Double-click anywhere on the map to place a numbered marker on an FB code — even if the pipeline missed it. Each marker is numbered in order. Toggle Marking Mode off to use the panel for panning only without accidentally placing markers.

Submitting a Sheet

Submit & Next Sheet — saves your markers and advances. At the end of a town it automatically moves to the next town (or state).

No Detections — confirms there are no FB codes on this sheet at all. Submits with 0 markers and a distinct status so it's distinguishable from an accidental empty submit.

Skip — moves on without recording anything. The sheet is not marked as reviewed.

Other Controls

Undo Last / Clear All — remove your most recent marker or all markers for the current sheet.

Unlink Panning — decouples the two panels so you can pan and zoom them independently. Re-click to sync them again.

Reset Zoom — returns both panels to the default full-sheet view.

Flag for Review — marks the sheet for expert follow-up and reveals a free-text notes field.

Redo Sheets — select previously submitted sheets to reopen and revise your answers.
